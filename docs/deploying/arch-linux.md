# Matron Server for Arch Linux

Currently Matron Server is only on the Arch User Repository (AUR).

The Matron Server AUR packages are community maintained and are not maintained by
Matron Server development team, but the AUR package maintainers are in the Matrix
room. Please attempt to verify your AUR package's PKGBUILD file looks fine
before asking for support.

- [matron-server](https://aur.archlinux.org/packages/matron-server) - latest tagged
matron-server
- [matron-server-git](https://aur.archlinux.org/packages/matron-server-git) - latest git
matron-server from `main` branch
- [matron-server-bin](https://aur.archlinux.org/packages/matron-server-bin) - latest
tagged matron-server static binary
